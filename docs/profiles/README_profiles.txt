Perfis de Tema (dual.infodose)

Arquétipo por tema (aliases → canônicos):
- NEON  → “Mago”     → Kaos (⚡) — rápido, grave leve
- LUX   → “Governante” → Atlas (⚙) — firme, pausado
- VIBE  → “Criador”  → Genus (🧩) — criativo, cadência suave
- DARK  → “Sábio”    → Solus (🕯) — baixo, introspectivo
- LIGHT → “Inocente” → Lumine (🌅) — claro, leve

Como usar:
1) Coloque a pasta 'profiles' ao lado do seu HTML.
2) No Theme Menu (botão do círculo), toque em “Perfis do Tema” para carregar o perfil do tema ativo.
3) O preset aplica theme + rate/pitch/volume + voice (se o player suportar).
