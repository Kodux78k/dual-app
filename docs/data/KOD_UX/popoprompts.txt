# Funções integradas que a página deve conter de forma real e ativa:

- **uploadHTML**: permite carregar arquivos HTML e exibir seu conteúdo no editor, aplicando realce de sintaxe.  
- **downloadHTML**: gera e baixa arquivo HTML baseado no conteúdo editado, preservando estrutura.  
- **downloadPromptBtn**: baixa o prompt base usado, para o usuário modificar e expandir offline.  
- **searchHTML**: busca instantânea com filtro em linhas do editor, atualizando conteúdo exibido.  
- **narrativeNav**: botão para navegar entre arquétipos 3, 6 e 9, mudando texto e ativando efeitos visuais e sonoros.  
- **mindSpace**: área clicável que alterna imagens fractais e mensagens motivacionais de ativação mental.  
- **player som ambiente**: botão liga/desliga som ambiente da Fórmula Dopamina Sex, com feedback visual animado.  
- **efeitos sonoros hover/click**: micro interações auditivas para todos botões padrão.  
- **fundo animado partículas**: canvas com partículas místicas se movendo em loop suave e responsivo.

# Como aplicar personalização via Infodose e usuários:

Na área lateral de prompt, adicionar um campo onde o usuário pode colar sua base de prompt da Infodose, que será concatenada e mesclada com o prompt principal — isso alimenta o motor com dados extras para geração ainda mais personalizada.

Também pode-se inserir preferências de cores, sons e temas, que refletem nas variáveis CSS e ativam variações visuais e auditivas, por exemplo:

```js
const userCustomization = {
  colors: { primary: "#00ffff", secondary: "#ff00ff", bg: "#000011" },
  promptExtra: "Base pessoal da Infodose: ...",
  soundVolume: 0.08,
  enableAnimations: true,
};
