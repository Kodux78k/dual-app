Welcome! This app simulates a tiny filesystem stored in your browser (localStorage). Use the Tabs below or jump straight into Terminal.

Quick Start

  1. Go to Terminal → tap ls, cd notes, then pwd.
  2. Open a file: open /notes/hello.txt or cat /samples/readme.txt.
  3. Browse files in Folders, long-press an item for actions.

Friendly Commands

Think of this like your phone’s private drawer. Big buttons run safe actions. Destructive actions ask before doing anything.

  • ls [path] — list a folder (default: current)
  • cd [path], cd .. — change directory
  • pwd — show current path
  • cat <file> — print file in terminal
  • open <file> — open in editor
  • touch <file> — create empty file
  • mkdir <folder> — create folder(s)
  • rm <path> [-r] — remove (ask to confirm)
  • mv <src> <dst> — move/rename
  • cp <src> <dst> — copy
  • download <file> — save as .txt (text only)
  • clear — clear terminal output
  • help [cmd] — quick guide
